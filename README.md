# GoSchool

GoSchool est une application web dynamique d'orientation scolaire et de comparaison d'ecoles.

## Stack principale

- Next.js App Router
- React
- TypeScript
- Tailwind CSS
- API Routes Next.js
- Backend Node.js / Express prevu
- PostgreSQL prevu pour la persistance

La version HTML dans `findschool/site/index.html` sert uniquement de prototype visuel ouvrable sans serveur. Le produit principal est dans `findschool/frontend`.

## Lancer l'application dynamique

```bash
cd findschool/frontend
npm install
npm run dev
```

Puis ouvrir `http://localhost:3000`.

Node.js `20.9.0` ou plus recent est recommande.
